# Dak Scan - Secure PDF Scanner Utility

**Dak Scan** is a high-performance, privacy-focused mobile application built with **Flutter & Dart**. It allows users to quickly scan documents, images, and notes, converting them into high-quality PDF files directly on their device.

## 🚀 Features

* **Smart Scanning:** Auto-detects document edges for perfect crops.
* **Enhancement Filters:** Improve scan quality with magic color, grayscale, and B&W filters.
* **PDF Creation:** Instantly convert multiple scans into a single PDF document.
* **Organize:** Manage scanned documents within the app with custom folders and names.
* **Share:** Easily share PDFs via email, WhatsApp, or cloud storage.
* **Offline First:** All processing happens locally on the device. No data is sent to external servers.

## 🛠️ Tech Stack

* **Frontend:** Flutter (Dart)
* **Local Storage:** Hive / SQLite (Choose whichever you used)
* **Image Processing:** `google_ml_kit` or similar packages used.
